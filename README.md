🌍 The Mission: Empowering the Next Billion

The core humanitarian objective is the global deployment of the "Oracle at Delphi in a box." This project involves distributing 100+ self-contained, solar-powered machines to children in regions without internet out-routes, specifically targeting communities across Africa.

* The Oracle Hardware: Solar-powered setups equipped with 2TB SSDs containing the "whole of humanity"—a massive, curated repository of human knowledge.
* The Ethical Obligation: AI must be "free as in freedom, free as in beer." High-level reasoning should not require a credit card or a high-bandwidth connection.
* The Synapse Bridge: The "Synapse" module is the technical cornerstone of this mission, providing interchangeable connectivity between Ollama (zero-bandwidth local inference) and OpenRouter (cloud failover). This ensures the "Oracle" remains operational and intelligent in the most isolated environments on Earth.

🧠 Are-Self: The Autonomous Reasoning Swarm

Are-Self is a brain-mapped architecture that enables small, free, local language models to function as a collaborative swarm. It provides the mechanical structure for consumer-grade hardware to perform autonomous tool use and long-term memory management.

Brain-Mapped Architecture

The system mirrors biological principles, mapping software modules to the specific functional responsibilities of the human brain.

Brain Region	Are-Self Component	Functional Responsibility
Hippocampus	Memory & Vector Search	Permanent storage of engrams/facts using vector-embedded similarity search.
Hypothalamus	Model & Budget Manager	Advanced parsing of model names, context windows, and failover circuit breakers.
Prefrontal Cortex	Task Management (PFC)	High-level assignment of Epics, Stories, and Tasks to the agentic swarm.
Central Nervous System	Execution Engine (CNS)	The core orchestration pathways and signal broadcasting.
Peripheral Nervous System	Worker Monitoring (PNS)	The system heartbeat and real-time monitoring of the worker fleet.
Frontal Lobe	Reasoning Sessions	Real-time monitoring of active reasoning logs and "Spike Trains."
Thalamus	Chat Interface	The unified sensory gateway/chat interface for user-system interaction.
Temporal Lobe	Iteration Controller	Management of work cycles, shifts, and participant turn limits.
Parietal Lobe	Tool Gateway	The execution bridge for the AI to interact with external environments.
Corpus Callosum	Self-Analysis	Specialized reflective module for the network to analyze its own activity.
